CSS Box Model & Spacing

The CSS Box Model is the foundation of layout and spacing in web design. Every HTML element can be thought of as a rectangular box with layers: content, padding, border, and margin.

1. Box Model Structure

/* Example */
div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Output:

Content Area

2. Padding

Padding is the space between the content and the border of an element.

div {
  padding: 30px;
}

Output:

This box has large padding.

3. Margin

Margin is the space outside the border of an element.

div {
  margin: 40px;
}

Output:

This box has a big margin.

4. Border

The border wraps around the padding and content.

div {
  border: 3px dashed red;
}

Output:

Dashed border around content.
