1. CSS Syntax

  property: value;
  property: value;
}

Example:

<style>
  body { background: #f7f7f7; font-family: Arial; }
  .card { background: white; padding: 12px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.05); }
  .card h3 { color: #222; margin: 0 0 8px; }
</style>

<div class="card">
  <h3>Styled Card</h3>
  <p>This is a simple CSS example.</p>
</div>

2. Selectors & Box Model

/* selector examples */
h1 { font-size: 28px; }
#main { width: 800px; margin: 0 auto; }
.container p { color: #555; }

/* box model: padding, border, margin */
.box { padding: 10px; border: 1px solid #ddd; margin: 10px 0; }
