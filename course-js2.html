JavaScript 101 - Variables

In JavaScript, variables are used to store data values such as numbers, text, or more complex data. You can declare variables using var, let, or const.

Types of Variables:

  • var – Old way of declaring variables (function-scoped).
  • let – Modern way for variables that can change (block-scoped).
  • const – For values that should never change (block-scoped).

Example:

<!DOCTYPE html>
<html>
  <body>
    <script>
      let name = "Mash";
      const year = 2025;
      var age = 20;

      document.write("Name: " + name + "<br>");
      document.write("Year: " + year + "<br>");
      document.write("Age: " + age);
    </script>
  </body>
</html>

Output:

Name: Mash
Year: 2025
Age: 20
