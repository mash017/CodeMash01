CSS Positioning

CSS Positioning allows you to control exactly where elements appear on the page. The main position values are:

  • static – Default position, elements appear in normal flow.
  • relative – Positioned relative to its normal position.
  • absolute – Positioned relative to the nearest positioned ancestor.
  • fixed – Positioned relative to the viewport and doesn’t move when scrolling.
  • sticky – Switches between relative and fixed depending on scroll position.

Example

.box {
  position: absolute;
  top: 50px;
  left: 100px;
  background-color: lightblue;
  padding: 10px;
}
