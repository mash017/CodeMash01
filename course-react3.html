State in React

State is a built-in object in React that allows components to create and manage their own data. When the state changes, the component automatically re-renders.

Example: Using State

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>Increase</button>
    </div>
  );
}

export default Counter;

Output:

Count: 0
(Click "Increase" to update the count)

Key Points

  • State is managed within the component
  • Changing state triggers a re-render
  • useState is a Hook used for managing state in functional components
