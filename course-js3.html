JavaScript 101 - Data Types

JavaScript has several built-in data types that define the kind of values you can store in variables.

Common Data Types:

  • String – Text values ("Hello")
  • Number – Numeric values (42, 3.14)
  • Boolean – Logical values (true or false)
  • Array – List of values ([1, 2, 3])
  • Object – Key-value pairs ({name: "Mash", age: 20})

Example:

<!DOCTYPE html>
<html>
  <body>
    <script>
      let name = "Mash"; // String
      let age = 20; // Number
      let isStudent = true; // Boolean
      let skills = ["HTML", "CSS", "JavaScript"]; // Array
      let profile = { name: "Mash", age: 20 }; // Object

      document.write("Name: " + name + "<br>");
      document.write("Age: " + age + "<br>");
      document.write("Student: " + isStudent + "<br>");
      document.write("Skills: " + skills.join(", ") + "<br>");
      document.write("Profile: " + profile.name + " (" + profile.age + ")");
    </script>
  </body>
</html>

Output:

Name: Mash
Age: 20
Student: true
Skills: HTML, CSS, JavaScript
Profile: Mash (20)
