HTML Basics

HTML (HyperText Markup Language) is the main language used to create web pages. It helps us build the structure of a web page. Below are some basic HTML tags:

1. HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

2. Common HTML Tags

  • <h1> ... <h6> : Heading tags, with h1 being the largest.
  • <p> : For paragraphs.
  • <a href="URL"> : To create links.
  • <img src="image.jpg" alt="description"> : To display images.
  • <ul>, <ol>, <li> : For creating lists.

Example Code

<!DOCTYPE html>
