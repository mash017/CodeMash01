CSS Backgrounds & Borders

CSS gives us control over the background colors, images, and borders of elements.

1. Background Color

div {
  background-color: lightblue;
}

Output:

This box has a light blue background.

2. Background Image

div {
  background-image: url('background.jpg');
  background-size: cover;
}

Output:

This box has a background image.

3. Borders

div {
  border: 2px solid black;
  border-radius: 10px;
}

Output:

This box has a solid border and rounded corners.

4. Gradients

div {
  background: linear-gradient(to right, red, yellow);
}

Output:

This box has a gradient background.
