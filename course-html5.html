HTML Images

Images help make web pages more engaging. In HTML, the <img> tag is used to embed images.

Basic Image Example

<img src="example.jpg" alt="Example Image" width="300">

Output:

Image with a Link

<a href="https://www.example.com">
  <img src="example.jpg" alt="Example Image">
</a>

Responsive Image

<img src="example.jpg" alt="Responsive Image" style="max-width:100%; height:auto;">
