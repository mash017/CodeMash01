HTML Semantic Elements

Semantic elements clearly describe their meaning in a human- and machine-readable way. They help structure your HTML document and improve accessibility and SEO.

Common Semantic Elements

  • <header> - Represents introductory content or navigation links.
  • <nav> - Contains the navigation menu.
  • <main> - The main content of the document.
  • <section> - Groups related content.
  • <article> - Independent, self-contained content.
  • <footer> - Contains footer information.

Example

<header>
  <h1>My Blog</h1>
</header>
<nav>
  <a href="#">Home</a> | 
  <a href="#">About</a>
</nav>
<main>
  <article>
    <h2>Blog Post Title</h2>
    <p>This is the content of the blog post.</p>
  </article>
</main>
<footer>
  <p>© 2025 My Blog</p>
</footer>
