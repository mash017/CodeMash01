CSS Colors & Text Styling

CSS provides many properties to style text and apply colors for visual impact and readability.

1. Colors

You can use color names, HEX codes, RGB, or HSL values.

h1 { color: red; }
p { color: #333; }
button { background-color: rgb(0, 0, 0); color: white; }

Output:

Red Heading

This is a paragraph with dark gray text.

2. Text Styling

h1 { font-family: Arial, sans-serif; font-size: 28px; font-weight: bold; }
p { font-style: italic; line-height: 1.6; letter-spacing: 1px; }

Output:

Styled Heading

This paragraph uses italic style, increased line height, and extra letter spacing.

3. Text Alignment

.center { text-align: center; }
.right { text-align: right; }

Output:

This is centered text.

This is right-aligned text.
