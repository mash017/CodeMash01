JavaScript 101 - Operators

Operators in JavaScript are used to perform actions on values and variables.

Types of Operators:

  • Arithmetic – +, -, *, /, %
  • Assignment – =, +=, -=
  • Comparison – ==, ===, !=, >, <
  • Logical – &&, ||, !

Example:

<!DOCTYPE html>
<html>
  <body>
    <script>
      let a = 10, b = 5;

      document.write("Addition: " + (a + b) + "<br>");
      document.write("Subtraction: " + (a - b) + "<br>");
      document.write("Multiplication: " + (a * b) + "<br>");
      document.write("Division: " + (a / b) + "<br>");
      document.write("Is a greater than b? " + (a > b) + "<br>");
      document.write("Both positive? " + ((a > 0) && (b > 0)));
    </script>
  </body>
</html>

Output:

Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2
Is a greater than b? true
Both positive? true
