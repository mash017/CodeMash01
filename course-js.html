JavaScript 101 - Introduction

JavaScript is a programming language that adds interactivity, logic, and dynamic features to web pages. It works alongside HTML and CSS to create engaging experiences.

What You Can Do With JavaScript:

  • Manipulate HTML content dynamically.
  • Respond to user actions (clicks, typing, scrolling).
  • Validate form inputs before submission.
  • Create animations and visual effects.
  • Fetch and display data from APIs.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript Example</title>
  </head>
  <body>
    <h1 id="demo">Hello!</h1>
    <script>
      document.getElementById("demo").innerText = "Hello, JavaScript!";
    </script>
  </body>
</html>

Output:

Hello, JavaScript!
