CSS Text Styling

CSS provides various properties to style text, such as changing fonts, size, color, spacing, and alignment.

1. Changing Text Color & Font

p {
  color: blue;
  font-family: Arial, sans-serif;
}

Output:

This is blue text with Arial font.

2. Font Size

h1 {
  font-size: 36px;
}
p {
  font-size: 18px;
}

Output:

Large Heading

Regular paragraph text.

3. Text Alignment

h2 {
  text-align: center;
}

Output:

Centered Text

4. Text Decoration

a {
  text-decoration: none;
  color: red;
}

Output:

Styled Link

5. Text Transform

p {
  text-transform: uppercase;
}

Output:

this text is uppercase
