JavaScript 101 - Conditional Statements

Conditional statements allow you to execute certain blocks of code based on a condition.

Types of Conditional Statements:

  • if – Runs a block if the condition is true.
  • if...else – Runs one block if the condition is true, another if false.
  • else if – Checks multiple conditions.

Example:

<!DOCTYPE html>
<html>
  <body>
    <script>
      let marks = 85;

      if (marks >= 90) {
        document.write("Grade: A+");
      } else if (marks >= 80) {
        document.write("Grade: A");
      } else if (marks >= 70) {
        document.write("Grade: B");
      } else {
        document.write("Grade: C or below");
      }
    </script>
  </body>
</html>

Output:

Grade: A
