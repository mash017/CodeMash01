Forms and Tables

Forms are used to collect user input, and tables are used to display data in a structured way.

1. HTML Form Example

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Submit">
</form>

Output:







2. HTML Table Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
    <td>UK</td>
  </tr>
</table>

Output:

Name Age Country
Alice 25 USA
Bob 30 UK
