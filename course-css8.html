CSS Flexbox

Flexbox is a CSS layout module that makes it easy to design flexible and responsive layouts. It helps distribute space and align items within a container.

Main Properties:

  • display: flex; – Enables flexbox on a container.
  • flex-direction – Row or column layout.
  • justify-content – Aligns items horizontally (start, center, space-between).
  • align-items – Aligns items vertically.
  • flex-wrap – Wraps items onto multiple lines.

Example

.container {
  display: flex;
  justify-content: space-between;
  background-color: #ddd;
}

.item {
  background-color: lightcoral;
  padding: 20px;
  margin: 5px;
  color: white;
}
