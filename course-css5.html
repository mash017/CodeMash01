CSS Backgrounds & Borders

CSS allows you to add backgrounds and borders to elements to improve design and structure.

1. Background Color

div {
  background-color: lightblue;
}

Output:

This div has a light blue background.

2. Background Image

div {
  background-image: url('example.jpg');
  background-size: cover;
}

(Image not displayed here; you can use any image URL.)

3. Border Styles

p {
  border: 2px solid black;
}

Output:

This paragraph has a black border.

4. Rounded Borders

div {
  border: 2px solid green;
  border-radius: 10px;
}

Output:

Rounded border example.
