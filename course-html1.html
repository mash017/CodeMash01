HTML Lists

HTML provides two main types of lists to organize content:

  • Ordered List (<ol>): Displays items in a numbered sequence.
  • Unordered List (<ul>): Displays items with bullet points.
  • Definition List (<dl>): Defines terms and descriptions.

1. Ordered List Example

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
  1. First item
  2. Second item
  3. Third item

2. Unordered List Example

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
  • Apple
  • Banana
  • Cherry

3. Definition List Example

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets
