HTML Tables

Tables allow you to display data in rows and columns. They consist of table rows, headers, and data cells.

1. Basic Table Structure

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
  </tr>
</table>
Name Age
Alice 25
Bob 30

2. Table with Caption

<table>
  <caption>User Info</caption>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>
User Info
Name Age
Alice 25
