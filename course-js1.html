JavaScript 101

JavaScript is a programming language used to create dynamic and interactive elements on web pages. It works alongside HTML and CSS to enhance user experience.

Example:

<script>
    function greet() {
        alert("Hello, welcome to JavaScript 101!");
    }
</script>

<button onclick="greet()">Click Me</button>
