HTML Links

Links are one of the most important elements of the web. They allow users to navigate between pages and resources. In HTML, links are created with the <a> tag.

Basic Link Example

<a href="https://www.example.com">Visit Example</a>

Output:

Visit Example

Opening Links in a New Tab

<a href="https://www.example.com" target="_blank">Open in new tab</a>

Email Links

<a href="mailto:someone@example.com">Send Email</a>
